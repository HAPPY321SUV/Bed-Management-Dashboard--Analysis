# Bed Management Dashboard Analysis

## Project Overview
Developed an interactive Power BI dashboard to monitor hospital bed occupancy, allocation efficiency, and capacity utilization.

## Tools Used
- Power BI
- Excel
- DAX
- Power Query

## KPIs
- Total Beds
- Occupied Beds
- Available Beds
- Occupancy Rate %
- Bed Turnover Rate

## Visualizations
- Occupancy Trend
- Department-wise Bed Utilization
- Bed Allocation Analysis
- Capacity Monitoring
